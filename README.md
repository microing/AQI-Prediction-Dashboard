A full-stack Air Quality Monitoring Dashboard built using React, TypeScript, Django, and OpenWeather API.

## Features

* Real-time AQI Search
* AQI Monitoring for 29 Indian State Capitals
* Search History Tracking
* Pollution Insights Dashboard
* Responsive UI
* Live Deployment

## Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Axios

### Backend

* Django
* Django REST Framework
* SQLite

### APIs

* OpenWeather Air Pollution API

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
python manage.py runserver
```
